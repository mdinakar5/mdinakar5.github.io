#linear
def gradient_descent(X, y, T, alpha): 
    loss = numpy.zeros(T)   # loss over time
    a, b = X.shape       # a is the # of examples, b is the # of features
    theta = numpy.zeros(b)  # initialize parameters
    
    for i in range(T):  #iterates num_steps times
        loss[i] = 0.5*numpy.linalg.norm(X.dot(theta) - y)**2 # cost for current parameter 
        grad = X.T.dot(X.dot(theta) - y) #compute ascent with highest slope
        theta = theta - alpha*grad #go down the gradient
    return theta, loss

Part 2 Problem 2¶

Derivation:

A generic loss function, C (sum of each cost): $$C = \sum_{i=1}^{n} cost(p_i(\beta^k), y_i)$$

y would either be 0 or 1, so new loss function: $$ cost(p(\beta^k), y) = -ylog(p(\beta^k)) - (1-y)log(1-p(\beta^k))$$

The logit function: $$p_i(\beta^k) = \frac{e^{f_i(\beta^k)}}{1+e^{f_i(\beta^k)}} * \frac{1+e^{-f_i(\beta^k)}}{1+e^{-f_i(\beta^k)}} = \frac{e^{f_i(\beta^k)}+1}{(e^{f_i(\beta^k)}+1)(1+e^{-f_i(\beta^k)})} = \frac{1}{(1+e^{-f_i(\beta^k)})} $$

The loss function: $$-\sum_{i=1}^{n} y log(p(\beta^k)) + (1-y)log(1-p(\beta^k))$$

The derivative of the logit: $$ p(\beta^k)(1-p(\beta^k)) * f'(\beta^k) $$

To minimize the convex cost, the gradient for linear regression will be used. L = argument of the summation: $$L(\beta^k) = y log(p(\beta^k)) + (1-y)log(1-p(\beta^k))$$ $$ \frac{\partial}{\partial \beta^k}L(\beta^k) = y * \frac{1}{p(\beta^k)} * p'(\beta^k) + (1-y) * \frac{1}{1-p(\beta^k)} * -p'\beta^k $$ $$ = \bigg(\frac{y}{p(\beta^k)} - \frac{1-y}{1-p(\beta^k)}\bigg)p'\beta^k $$

$$ = \bigg(\frac{y}{p(\beta^k)} - \frac{1-y}{1-p(\beta^k)}\bigg)(p(\beta^k)(1-p(\beta^k)) * f'(\beta^k))$$$$ = \Big(y(1-p(\beta^k)) - (1-y)(p(\beta^k))\Big)x_i$$$$ = (y - p(\beta^k))x_i$$

Using the gradient, the parameters are updated. The equation: $$ \theta_{n+1} = \theta{n} - \alpha \cdot \nabla $$ Since there is a negative outside of the summation, there is a double negative which cancels. New equation: $$\beta^{k+1} = \beta^{k} + \alpha\sum_{i=1}^{n}(y_i - p_i(\beta^k))x_i$$

#logistic batch version
#p_i(b^k)
def logit(x):
    return (numpy.exp(x)/(1+numpy.exp(x)))

def log_gradient_descent(X, y, T, alpha):
    loss = numpy.zeros(T)  # loss over time
    a, b = X.shape # a is the # of examples, b is the # of features
    theta = numpy.zeros(b) 
    
    for i in range(T): #iterates num_steps times
        res = logit(X.dot(theta))
        loss[i] = (numpy.log(1+numpy.exp(res.dot(y)))) # cost for current parameter 
        res2 = X.T.dot(res-y) #compute ascent with highest slope
        theta = theta + alpha*res2 #go down the gradient
    return theta, loss
